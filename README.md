# Random Quote Generator — Генератор случайных цитат

Графическое приложение для генерации случайных цитат с фильтрацией по автору
и теме, сохранением истории и тестами.

**Автор: Андерсон Богдан**

---

## Возможности

- База из 20+ предопределённых цитат (текст, автор, тема)
- Кнопка «Сгенерировать цитату» — выбирает случайную с учётом фильтра
- Фильтрация по автору и по теме
- Кнопка «Добавить свою цитату» с валидацией (пустые поля не принимаются)
- История сгенерированных цитат в виде списка
- Сохранение истории в `history.json` и загрузка при следующем запуске

---


## Установка и запуск

### Windows

```bash
python main.py
```

### Linux

```bash
sudo apt install python3-tk
python3 main.py
```

---

## Запуск тестов

```bash
python -m unittest tests -v
```

12 тестов покрывают:
- корректность базы цитат
- логику фильтрации
- валидацию при добавлении пользовательской цитаты
- сохранение / загрузку JSON

---

## Примеры использования

1. Запустить `python main.py`
2. Выбрать фильтр по автору или теме (или оставить «Все»)
3. Нажать **Сгенерировать цитату** — цитата отобразится в поле
4. История всех генераций сохраняется внизу
5. Нажать **Добавить свою цитату** — ввести текст, автора, тему
